##### 任务简报

给你一个数组 `arr`，把每个元素替换成它排序后的名次。  
规则：数字越大名次越大，相同数字名次相同，名次从1开始，且尽可能小。

案例怪物：`[40, 10, 20, 30]`  
排序去重后的实力榜：`[10, 20, 30, 40]` → 名次 `[1, 2, 3, 4]`  
输出：`[4, 1, 2, 3]`

##### 一号选手·暴力一行流

好的，一位萌新选手率先冲了上去！他掏出了一卷极其华丽的代码卷轴！

```python
return [sorted(set(arr)).index(i) + 1 for i in arr]
```

观众们，一行流，极简美学！——但是，画面卡住了！加载圆圈转个不停，然后一个巨大的 **TLE** 砸在屏幕上，直接把萌新选手送回了复活点。

导播，麻烦给个慢镜头：

- `sorted(set(arr))` 放在列表推导式里，**每次循环都重新排列整个怪物图鉴**！  
- `.index(i)` 又是从图鉴第一个开始挨个点名，**线性大海捞针**。

这就好比你在副本里，每砍一只小怪，都要把整个地图的怪物重新清点一遍，再从地图入口跑到它面前确认位置。时间复杂度爆炸成 `O(n² log n)`——BOSS还没见到呢，先把自己跑死了。

**结论**：这卷轴是诅咒道具，捡到请直接销毁。

##### 二号选手·哈希圣光

接下来上场的是经验丰富的老玩家。他抬手就是一个 **哈希映射·圣光护符**！

```python
def arrayRankTransform(arr: List[int]) -> List[int]:
    sorted_unique = sorted(set(arr))               			# 一次排序，世界清净
    rank_map = {v:i for i,v in enumerate(sorted_unique,1)}  # 人手一张名次卡
    return [rank_map[v] for v in arr]              			# 瞬移贴标签，德芙都没我丝滑
```

观众们，看这波操作！  他先花 `O(n log n)` 的代价，一次性把怪物实力榜排好，给每个怪物发了名次卡片。然后拿着卡片直接 **O(1)** 瞬移贴标签，全程没有被诅咒摸到一下，完美通关！

**数据面板：**

- 排序耗时 `O(n log n)`  （一次性投资，终身受益）
- 建表耗时：`O(n)` （给每个怪物发身份证）
- 查表总耗时 `O(n)`  （O(1)查询就是爽）
- 总伤害 `O(n log n)`，BOSS直接融化，AC到手。

##### 三号选手·二分飞刀

最后登场的，是职业选手的操作秀——他从怀里掏出了一柄寒光闪闪的 **二分飞刀**！

```python
from bisect import bisect_left

def arrayRankTransform(arr: List[int]) -> List[int]:
    sorted_unique = sorted(set(arr))
    # 咻咻咻，二分飞刀连斩
    return [bisect_left(sorted_unique, val) + 1 for val in arr]
```

**看这飞刀轨迹！**——`bisect_left` 在有序列表里用折半查找，**每次定位都是 O(log n)**，砍掉一半搜索空间！虽然单次没圣光 O(1) 快，但这飞刀是C语言淬炼过的，常数因子极小，整套连招如德芙般纵享丝滑。

**同样 O(n log n) 的DPS**，但职业选手就是偏爱这种肌肉记忆般的微操。当你需要反复在同一个有序列表里搜索时，这招就是你的本命飞刀。
